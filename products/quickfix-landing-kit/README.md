# QuickFix Landing Kit

Three editable static landing page starters for small paid web jobs.

## Files

- `templates/saas.html` - SaaS/product landing page
- `templates/local-service.html` - local service business page
- `templates/creator.html` - creator/portfolio offer page
- `assets/kit.css` - shared styling
- `assets/kit.js` - tiny interaction helper

## How to use

1. Open one of the HTML files in a browser.
2. Replace the copy, links, colors, and contact details.
3. Upload the folder to GitHub Pages, Netlify Drop, Cloudflare Pages, or any static host.

No build step is required.
